
# PCCW Login Test

## Usage
```
git clone https://github.com/hollemeng/login_test.git
cd login_test
npm i
npm run cypress:justGO
```
## The report from last run is hosted in Github page: [https://hollemeng.github.io/login-test/](https://hollemeng.github.io/login-test/)
